# Lançamento Orçamentário

Base em React + Vite + Tailwind 3 para prototipar a tela de lançamento orçamentário.

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar arquivos para o Sankhya

```bash
npm run build
```

Depois, compacte o conteúdo de `dist` neste formato:

```text
index.html
assets/
```

Não compacte a pasta `dist` por fora.
